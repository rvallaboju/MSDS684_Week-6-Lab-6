# Lab 6 Notebook

This repository contains the Week 6 Lab 6 notebook for the MSDS 684 course. The notebook covers various data analysis techniques and showcases how to implement them using Python.

## Setup Process
1. **Clone the repository**:
   ```bash
   git clone https://github.com/rvallaboju/MSDS684_Week-6-Lab-6.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd MSDS684_Week-6-Lab-6
   ```

3. **Install required packages**:
   Make sure you have the necessary libraries installed. You can use the following command:
   ```bash
   pip install -r requirements.txt
   ```

## Installation Requirements
- Python 3.x
- Jupyter Notebook
- Libraries listed in requirements.txt, which include:
   - pandas
   - numpy
   - matplotlib
   - seaborn
   - scikit-learn

## Running the Notebook
To run the notebook, use the following command in your terminal:
```bash
jupyter notebook Vallaboju_Ruthika_Lab 6.ipynb
```
